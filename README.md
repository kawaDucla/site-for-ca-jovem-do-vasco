# Site da Força Jovem do Vasco

Projeto de site institucional da **Força Jovem do Vasco**, desenvolvido utilizando apenas **HTML e CSS**.
O objetivo do site é apresentar a identidade da torcida, suas informações principais e fortalecer a presença digital do movimento.

Este projeto tem caráter educacional e foi desenvolvido com foco em **frontend básico** e **estruturação de páginas web**.

---

## Funcionalidades

* Página institucional da Força Jovem do Vasco
* Seções informativas sobre a torcida
* Layout estático
* Estrutura sem backend
* Navegação simples e direta

---

## Tecnologias Utilizadas

* HTML5
* CSS3

---

## Como Executar o Projeto

1. Faça o download ou clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd forca-jovem-vasco
```

3. Abra o arquivo `index.html` no navegador
   ou utilize a extensão **Live Server** no VS Code para melhor experiência.
